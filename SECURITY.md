# Security Policy

  ## Supported Versions

  | Version | Supported          |
  | ------- | ------------------ |
  | 1.0.x   | :white_check_mark: |
  | < 1.0   | :x:                |

  ## Reporting a Vulnerability

  Please report security vulnerabilities via GitHub's private vulnerability reporting:
  https://github.com/egulatee/mcp-server-codecov/security/advisories/new

  **Please do not report security vulnerabilities through public GitHub issues.**

  We will respond within 48 hours and aim to release a fix within 7 days.

  ## Security Measures

  - Dependencies scanned via Dependabot
  - Code scanning with CodeQL
  - Secret scanning enabled
  - npm audit in CI/CD pipeline
  - Regular dependency updates
